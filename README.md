# Credit Risk Prediction

## Objective
To predict whether a loan applicant is likely to default using machine learning.

## Dataset
The dataset contains applicant details such as income, loan amount, education, credit history, and loan status.

## Approach
- Data cleaning (handled missing values)
- Exploratory Data Analysis (EDA) using visualizations
- Feature encoding using pandas
- Model training using Logistic Regression

## Results
- Model achieved good accuracy
- Confusion matrix used to evaluate predictions
- Visualizations helped understand key patterns in data

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Scikit-learn
